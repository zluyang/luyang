假作真时真亦假

发表于 2018-08-20   |   分类于 读书笔记   |     |   热度 ℃

梦

五点，梦中被奇怪的BGM乱入，过于荒谬，意识到是梦，醒来复盘，梦中竟然是小时候红白机上玩的游戏叫影子传说。
为啥会乱入这个游戏，想起小时候第一次玩的时候，一脸问号，两颗被辐射过的诡异大树下，一个女的站在屏幕中央咳嗽几下，脸都没露就被黑衣忍者带出了屏幕，然后把我load到game中，我下意识的追，以为那个该死的忍者肯定就在屏幕外不远处。。
一个大跳蹦老高，然后听到呼的一声，屏幕外僧侣的暗箭难防，蹦high了来不及降落。。GG
从新来过。。
第二次被load进来，冷静了一下，等等。。有什么地方不对，那个女的跟我啥关系，我为啥要顺着她被带走的方向追，我明显一身英雄扮相，那女人就是我娘子喽？那我娘子长得漂亮不？长得像凤姐我是不是就不要追了，我留在原地等他们从屏幕左边过来好不好，一路追多累，对，特么的不动了，发现特喵的不动不行，屏幕里一下多了三个敌人，本能地我只能扔镖举刀，屏幕里的杀干净了，继续等，特么的又来。。好吧，不逃避，追吧，伴随着诡异而让人焦虑的BGM，一路大跳、扔镖，终于救出了傻娘们，哈哈大笑，不过如此，发现游戏画面一转，又特么从头开始了，忍者又特么抢在我前面把傻娘们带走了，我觉得一定是按错了什么东西，导致跳过了关键的结尾动画，我必须证明自己是不是按错了，又是一路大跳、扔镖，再次带走傻娘们，嗯，这次什么都没多按，我看你怎么往下演，发现又是辐射的大树，无耻的忍着带走了傻娘们。

影子传说的设定荒诞吗？

西西佛斯是希腊神话中一位被惩罚的人。他受罚的方式是：必须将一块巨石推上山顶，而每次到达山顶后巨石又滚回山下，如此永无止境地重复下去。

普罗米修斯与智慧女神雅典娜共同创造了人类，普罗米修斯负责用泥土雕塑出人的形状，雅典娜则为泥人灌注灵魂，并教会了人类很多知识。当时宙斯禁止人类用火，他看到人类生活的困苦，帮人类从奥林波斯偷取了火，因此触怒宙斯。宙斯为了惩罚人类，将潘朵拉的盒子放到人间。再将普罗米修斯锁在高加索山的悬崖上，每天派一只鹰去吃他的肝，又让他的肝每天重新长上，使他日日承受被恶鹰啄食肝脏的痛苦。

希望自己总能第一时间意识到一些荒诞的设定，及时跳出循环，时常问问自己，娘子是谁？长得漂亮吗？我为什么要追？毕竟重复是最让人感到痛苦的惩罚。
