[WWDC2020 Sesstion 10099] Explore the Action & vision Demo

发表于 2020-09-12   |     |   热度 ℃

“Action & vision” Demo是综合使用Vision API 和 Create ML App 对人体动作进行分类的一个很好的例子。

Demo使用几种不同的投掷技术扔口袋，这些动作在应用程序中使用Create ML训练的Model进行分类，想详细了解这个demo的同学请移步这个Session
想详细了解Vision相关API的同学可以移步这个Session

Demo原理简析：

  • 使用VNCoreMLRequest用模型识别板子的位置
  • 一旦我们有了板子，可以使用VNTranslationalImageRegistrationRequest来分析场景稳定性，这也是保证正确性的先决条件
  • 用VNDetectContourRequest来对板子进行边缘检测画出板子轮廓View
  • 用VNDetectHumanBodyPoseQequest来找到投掷选手
  • 开始游戏后，用VNDetectTrajectoriesRequest追踪被投掷的球
  • 然后用一个新的训练模型分类我们投球姿势，据此进行打分，最后用轨迹分析来测量我们投掷的速度。
